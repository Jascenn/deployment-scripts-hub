问题 2: Docker 镜像拉取仍然失败
----------------------------------------
1. 打开 Docker Desktop
2. Settings → Docker Engine
3. 删除 "registry-mirrors" 配置
4. 点击 "Apply & Restart"
5. 重新运行 docker-deploy.bat

或者运行:

  fix-docker-mirrors.bat
