========================================
  欢迎使用 BettaFish Windows 部署工具
  版本 v4.0.3
========================================

🎯 快速开始（3步部署）
========================================

1️⃣  确保 Docker Desktop 正在运行

2️⃣  双击运行: docker-deploy-v4.bat

3️⃣  按提示输入 API 密钥

部署完成后自动打开: http://localhost:5000

========================================
  🔧 常用工具
========================================

部署工具:
  docker-deploy-v4.bat    一键完整部署（推荐）
  download-project.bat    下载项目源码
  quick-fix.bat           预拉取镜像

诊断工具:
  debug-env.bat           API 密钥诊断
  diagnose.bat            系统全面诊断

修复工具:
  fix-docker-mirrors.bat  修复镜像源问题
  fix-all.bat             修复编码问题
