## 🆘 常见问题

### Q: 这些文件要放在哪里？
**A:** 保持在当前位置即可，相对路径都是正确的。

### Q: 可以离线使用吗？
**A:** ✅ 完全可以！所有 HTML 和 Markdown 文件都是独立的，不需要网络。

### Q: macOS 和 Linux 有区别吗？
**A:** 基本相同，主要区别：
- macOS 用 `open` 命令打开文件
- Linux 用 `xdg-open` 或浏览器命令
- 包管理器不同（Homebrew vs apt/yum）

### Q: 需要安装额外软件吗？
**A:** 只需要浏览器（查看 HTML）和文本编辑器（查看 MD），都是系统自带的。
